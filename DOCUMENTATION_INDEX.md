## 🎯 Quick Reference

### **System Requirements**
- Node.js v14+
- npm or yarn
- MongoDB
- Gmail account

### **Tech Stack**
- Backend: Node.js, Express, MongoDB, Socket.io, Node-Cron
- Frontend: React, Vite, Socket.io-client
- Styling: CSS3 with variables

### **Key Features**
1. Stock management ✅
2. Auto stock deduction ✅
3. Email alerts ✅
4. Order status tracking ✅
5. Real-time updates ✅
6. Admin dashboard ✅

### **Port Numbers**
- Backend API: 5000
- Frontend Dev: 5173

### **Important Thresholds**
- Low stock alert: 20 units
- Hourly check: Every hour
- Daily report: 8:00 AM
